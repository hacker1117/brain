Henry 询问当前 OpenClaw 用的搜索工具，并主动考虑付费升级。以下是他在这次对话中形成的判断：

**当前状态：** DuckDuckGo 免费后端，不稳定。

**选型方向（Henry 认可的优先级）：**
1. **Perplexity**（首选）— 付费，适合主搜索，质量最高
2. **Exa**（补充）— 增强研究场景再加
3. **Tavily**（最后）— 做自动化 pipeline 时再上

**关于 Google 的判断：**
> "是 Google 没有提供对应的 API 吗？还是我没有装插件？还是说由于某种原因导致其实 Google 并不好用？"
