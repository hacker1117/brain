# 搜索工具选型：Perplexity 优先，Exa 次之，Tavily 做自动化

Henry 询问当前可用搜索工具及区别后，决定采购并配置：

- **当前默认**：DuckDuckGo 免费后端，不稳定
- **推荐配置顺序**：
  1. **Perplexity** — 主搜索，自然语言查询，回答质量高，付费 API
  2. **Exa** — 增强研究，语义搜索，适合文献/深度调研
  3. **Tavily** — 自动化 pipeline，结构化结果，适合 agent 流程

- **Google / Gemini**：OpenClaw 支持，但 Gemini + Search grounding 不是传统搜索 API；推荐 Perplexity 做主搜索而非 Google

Henry 已购买 Perplexity API key 并要求配置。Exa 注册遇到 Google 账号报错问题暂未解决。
