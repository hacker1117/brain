Henry 要求飞书 CLI 接入前先确认官方来源并阅读文档：

> “好的，那我们就先来做飞书的CLI工具。他们应该自己是开放了对应的CLI工具的，而且好评应该是很多的。所以你先去互联网上搜索一下，确保你拿到的是他们的官方文档。然后你大致地读一遍，再来告诉我我们应该怎么做。”

初步确认官方 CLI 是 `@larksuite/cli` / `lark-cli`，由 GitHub `larksuite/cli` 团队维护，npm package 描述为 “The official CLI for Lark/Feishu open platform”。
