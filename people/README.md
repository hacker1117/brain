# people/ — Person Pages

## What goes here
Individual people. Each page is one human. Use primary name as filename slug.

## What does NOT go here
- Companies (→ companies/)
- Concepts (→ concepts/)
- Meetings (→ projects/ or sources/)

## Page naming
`first-last.md` — e.g., `elon-musk.md`
