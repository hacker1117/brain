# 国内 AI 培训中的 Agent 工具选择

## 问题

面向中国制造业一线员工做 AI 实操培训时，理想工具与现实工具之间存在落差。

理想上，Agent 类型工具最好是 Claude Code 或 Manus 这类能力更强、交互更成熟的工具；但在真实培训现场，会受到这些约束：

- 学员电脑配置较低；
- VPN 配置困难；
- 海外模型账号配置困难；
- 学员需要低门槛、可当场安装、可当场完成任务的工具。

[Source: User, Telegram, 2026-05-18]

## GWC 现场经验

Henry 在 GWC 培训中最终选用了字节的 Trae Solo，原因是它更适合国内环境和现场安装。但实际体验显示：Trae Solo 开发还不完整，虽然模仿 Claude Desktop App，但在 Skill 配置、模型配置、产出文件的查看与下载上都不够好用。

[Source: User, Telegram, 2026-05-18]

## 后续观察点

之后做类似培训时，需要持续关注国内 Agent 工具是否出现一个更合适的窗口：

- 方便使用；
- 容易交互；
- 国内可访问；
- 对低配置电脑友好；
- 文件查看、下载、导出链路稳定；
- 模型与 Skill 配置足够简单。
